Генерация матрицы

Справка

Классическое умножение:

Матрица степени N показывает количество путей длины N между вершинами.
0 - пути нет, другое значение - количество путей.

Логическое умножение:

Матрица степени N показывает достижимость между вершинами по путям длины N.
1 - путь есть, 0 - пути нет.

Тропическое умножение:

Матрица степени N показывает максимальный вес путя длины N между вершинами.
0 - пути нет, другое значение - максимальный вес пути.
